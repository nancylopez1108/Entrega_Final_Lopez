Clinica Mateos: Es una web realizada en Django.
Para poder correr el trabajo usar python manage.py runserver. 
Ya dentro de la web, te encontrás con la página de inicio donde se visualiza la bienvenida a la página y arriba podés ir a la página que necesites,registrarte o iniciar sesion. 
Turnos: acá podés sacar un turno o buscar el turno que ya tenías reservado mediante tu DNI siempre y cuando estes logeado con usuario y clave
Médicos: muestra los médicos que hay disponibles en la clínica, si sos un super usuario podes eliminar o editar los datos del medico.
Otra función que esta disponible es la de editar tu usuario y agregar un avatar.
